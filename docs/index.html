registro

Consulta el endpoint de registro. Agrega un usuario a la tabla Usuario de la base de datos

registro(email: string, password: string, nombre: string, role: string): json
Parameters
email (string)
password (string)
nombre (string)
role (string) Debe tomar los valores ADMIN y USER
Returns
json: {email:email, nombre:nombre, role:role} o bien la palabra "duplicado" en caso que el usuario ya exista

login

Consulta el endpoint de login. Crea una cookie de sesión autenticada en el servidor

login(email: string, password: string): json
Parameters
email (string)
password (string)
Returns
json: {email:email, nombre:nombre, role:role} si el usuario no está registrado se devuelve la cadena "usuario no existe" si el password es incorrecto, se devuelve la cadena "password incorrecto"

obtenerPeliculasCatalogo

Se obtienen todas las películas guardadas en la tabla PeliculaCatalogo No requere parámetros; sólo puede consultarse por usuarios logueados

obtenerPeliculasCatalogo(): json
Returns
json: de la forma {"results":[propspeli1, propspeli2....]}

agregarPeliculaCatalogo

agregarPeliculaCatalogo(idPelicula: number, poster_path: string, title: string, release_date: string, original_language: string, vote_average: string, overview: string): json
Parameters
idPelicula (number)
poster_path (string)
title (string)
release_date (string)
original_language (string)
vote_average (string)
overview (string)
Returns
json: si la película fue agregada exitosamente, devuelve un joson con los parámetros {idPelicula, poster_path, title, release_date, original_language, vote_average, overview}

en caso contrario, devuelve la cadena "duplicada"

borrarPeliculaCatalogo

Borra una película del católogo

borrarPeliculaCatalogo(idPelicula: number): json
Parameters
idPelicula (number)
Returns
json: con los parámetros {estado:borrada,idPelicula} en la clave

obtenerPeliculasAlquiladas

Se obtienen las películas alquiladas para un usuario. El servidor se encarga de determinar, cuál es el usuario mediante la cookie JSESSION enviada desde el navegador cliente

obtenerPeliculasAlquiladas(): json
Returns
json: de la forma {"results":[propspeli1, propspeli2....]}

agregarPeliculaAlquilada

agregarPeliculaAlquilada(idPelicula: number): json
Parameters
idPelicula (number)
Returns
json: si se guarda exitosamente, de la forma { "estado": "guardada", "idPelicula": 1 }

de lo contrario devuelve una cadena que dice "duplicada"

borrarPeliculaAlquilada

Borra una película alquilada, por su idAlquilada, según el usuario logueado

borrarPeliculaAlquilada(idAlquilada: number): json
Parameters
idAlquilada (number) la cual es diferente al idPelicula
Returns
json: de la forma { "estado": "borrada", "idAlquilada": 1 }
